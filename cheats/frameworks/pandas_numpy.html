pandas & NumPy Cheatsheet

NumPy — Arrays & Basics

  • Import
    import numpy as np
  • Create array
    a = np.array([1,2,3,4])
  • Arithmetic ops
    b = a * 2     # element‑wise multiply
c = a + b           # element‑wise add
  • Matrix / 2D array
    m = np.array([[1,2,3],
                  [4,5,6]])
print(m.shape)  # (2,3)

pandas — DataFrame & Series

  • Import & DataFrame
    import pandas as pd
df = pd.DataFrame({
  'name': ['Alice','Bob','Charlie'],
  'age': [25, 30, 35]
})
  • Read / Write CSV
    df = pd.read_csv('data.csv')
df.to_csv('out.csv', index=False)
  • Info & Describe
    df.info()
df.describe()
  • Filter / Select
    df2 = df[df.age > 30]
names = df['name']
  • Group / Aggregation
    df.groupby('age').count()
df['age'].mean()

Advanced / Combined Usage

  • NumPy → pandas conversion
    arr = np.random.rand(5,3)
df = pd.DataFrame(arr, columns=['A','B','C'])
  • pandas read/write SQL via SQLAlchemy
    from sqlalchemy import create_engine
engine = create_engine('sqlite:///mydb.db')
df.to_sql('table_name', engine, if_exists='replace', index=False)
df2 = pd.read_sql_table('table_name', engine)
