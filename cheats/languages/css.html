CSS Cheatsheet

Basic

  • Selectors: `element`, `.class`, `#id`, `tag.class`
  • Box model: 
    
div {
  width: 200px;
  padding: 10px;
  border: 1px solid #333;
  margin: 20px;
}
  • Flexbox (container): 
    
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
  • Flexbox (child): 
    
.child {
  flex: 1; /* grow to fill space */
}
